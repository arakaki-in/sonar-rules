This rule detects repeated string concatenation using the + or += operators inside a loop.

Because Python strings are immutable, concatenating strings creates a new string object and copies the contents of both operands. Inside a loop, this leads to $O(N^2)$ time complexity. For optimal performance, append strings to a list inside the loop and use "".join() at the end.

Noncompliant Code Example

result_str = ""
for line in lines:
    result_str += line  # Noncompliant

Compliant Solution

parts = []
for line in lines:
    parts.append(line)
result_str = "".join(parts)  # Compliant
