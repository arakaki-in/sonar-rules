This rule detects membership checks like key in dict.keys().

Checking membership against dict.keys() creates unnecessary function call overhead. Checking membership directly in the dictionary key in dict is cleaner, faster, and operates in O(1) time.

Noncompliant Code Example

if "key" in my_dict.keys():  # Noncompliant
    print("Found")

Compliant Solution

if "key" in my_dict:  # Compliant
    print("Found")
