This rule detects the use of map(lambda ...) or filter(lambda ...).

Using map or filter with a lambda expression is often harder to read than a list comprehension or a generator expression. Furthermore, list comprehensions are typically faster because they avoid the function call overhead of the lambda for each element.

Noncompliant Code Example

squares = map(lambda x: x**2, numbers)  # Noncompliant
evens = filter(lambda x: x % 2 == 0, numbers)  # Noncompliant

Compliant Solution

squares = [x**2 for x in numbers]  # Compliant
evens = [x for x in numbers if x % 2 == 0]  # Compliant
