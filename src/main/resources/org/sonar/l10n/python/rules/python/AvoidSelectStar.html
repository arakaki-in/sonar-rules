This rule detects the use of SELECT * in SQL queries.

Using SELECT * pulls all columns from the database tables. This increases the amount of data transferred over the network and increases database I/O, particularly on tables with heavy or unused columns. Always project only the columns actually required by the application.

Noncompliant Code Example

query = "SELECT * FROM users"  # Noncompliant
cursor.execute(query)

Compliant Solution

query = "SELECT id, name FROM users"  # Compliant
cursor.execute(query)
