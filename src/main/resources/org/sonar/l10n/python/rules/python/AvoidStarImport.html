This rule detects the use of wildcard imports (e.g. from module import *).

Wildcard imports pollute the local namespace, make it difficult for static analysis tools to trace names, and can introduce silent bugs if imported modules change their exports. Explicitly import only the objects needed.

Noncompliant Code Example

from math import *  # Noncompliant
print(sin(0))

Compliant Solution

from math import sin  # Compliant
print(sin(0))
