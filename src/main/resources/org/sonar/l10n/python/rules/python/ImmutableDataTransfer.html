This rule ensures that data passed to threads, processes, or shared queues is immutable.

Passing mutable objects (like lists, dicts, or sets) across thread or process boundaries creates shared mutable state, resulting in data races, memory corruption, and synchronization issues. Data transferred between execution environments should be represented as immutable structures like tuples, frozensets, or frozen dataclasses.

Noncompliant Code Example

import threading
import queue

q = queue.Queue()
q.put([1, 2, 3])  # Noncompliant

t = threading.Thread(target=worker, args=([1, 2],))  # Noncompliant

Compliant Solution

import threading
import queue

q = queue.Queue()
q.put((1, 2, 3))  # Compliant (Tuple)

t = threading.Thread(target=worker, args=((1, 2),))  # Compliant
