This rule detects the use of type(x) == SomeType or type(x) is SomeType for type checking.

Using type() for type comparison does not account for inheritance — it only matches the exact type, not subclasses. The built-in isinstance() function properly handles inheritance hierarchies and is the Pythonic way to perform type checks. It also supports checking against multiple types via a tuple of types.

Noncompliant Code Example

if type(x) == int:  # Noncompliant
    process(x)

if type(x) is str:  # Noncompliant
    process(x)

Compliant Solution

if isinstance(x, int):  # Compliant
    process(x)

if isinstance(x, str):  # Compliant
    process(x)
