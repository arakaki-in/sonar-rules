This rule checks for string formatting using .format() or the % operator.

Python's formatted string literals (f-strings) are cleaner, more readable, and faster than older formatting methods (.format() or %), because they are evaluated at runtime rather than parsed dynamically.

Noncompliant Code Example

name = "Alice"
greeting = "Hello, {}".format(name)  # Noncompliant
greeting2 = "Hello, %s" % name  # Noncompliant

Compliant Solution

name = "Alice"
greeting = f"Hello, {name}"  # Compliant
