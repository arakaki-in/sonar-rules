This rule checks that high-volume data classes (e.g. classes ending in Model, DTO, Payload, Event, etc., or dataclasses) define __slots__.

By default, Python class instances have a dictionary __dict__ to store their attributes. This dictionary consumes a significant amount of memory. Defining __slots__ reserves space for the declared attributes and prevents __dict__ creation, dramatically reducing memory usage and slightly improving attribute access speed. For standard Python dataclasses, you can use @dataclass(slots=True).

Noncompliant Code Example

from dataclasses import dataclass

@dataclass
class UserDTO:  # Noncompliant (lacks __slots__)
    id: int
    name: str

Compliant Solution

from dataclasses import dataclass

@dataclass(slots=True)
class UserDTO:  # Compliant
    id: int
    name: str
