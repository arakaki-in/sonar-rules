This rule detects the use of standard library json.loads() and json.dumps().

For high-performance JSON operations, third-party libraries like orjson or ujson are significantly faster than Python's built-in json library. They are written in C or Rust and offer substantial speed improvements, particularly for large payloads or complex nested structures.

Noncompliant Code Example

import json

data = json.loads(payload)  # Noncompliant

Compliant Solution

import orjson

data = orjson.loads(payload)  # Compliant
