# Security Policy

## Supported Versions

We actively support and patch the latest release of the plugin.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please **do not** open a public issue. Instead, report it privately to the maintainer:

* **Contact Email**: `eveuca@gmail.com`

Please include the following details in your report:
* A description of the vulnerability and its potential impact.
* Detailed steps to reproduce the vulnerability (including mock code or environment configuration, if applicable).
* Any potential mitigation strategies or fixes you recommend.

We will acknowledge receipt of your report within 48 hours and work with you to investigate and resolve the issue. Thank you for helping keep this project secure!
